StudyCord

StudyCord is a full-stack collaborative learning web application built with Django.
It provides a platform where students can create and join study rooms, participate in topic-based discussions,
and collaborate with peers in an organized community-driven environment.

Features
User Registration and Authentication
User Login and Logout
Profile Management
Profile Picture Upload and Update
Study Room Creation and Management
Topic-Based Discussion Rooms
Room-Based Messaging System
Search Functionality
Activity Feed
Account Management
Mobile-Responsive Design
Clean and Intuitive User Interface

Installation
Clone the Repository
git clone https://github.com/amaan29/StudyCord.git
cd StudyCord
Create and Activate Virtual Environment
python -m venv venv


Windows:
venv\Scripts\activate
Install Dependencies
pip install -r requirements.txt
Run Database Migrations
python manage.py migrate
Start the Development Server
python manage.py runserver

Open your browser and visit:

http://127.0.0.1:8000/
